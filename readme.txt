# 👚 Wellnes - Tienda de Ropa

Bienvenida/o a **Wellnes**, una tienda de ropa enfocada en el bienestar, la comodidad y el estilo.  
Este proyecto fue desarrollado como parte de una práctica de desarrollo web, con el objetivo de crear una experiencia de compra atractiva, moderna y funcional.

---

## 🌟 Características principales

- Página de inicio con presentación de la marca  
- Sección “Nosotros” con información de la filosofía de Wellnes  
- Catálogo de productos con diseño responsive  
- Formularios de contacto y suscripción  
- Navegación simple e intuitiva

---

## 🛠️ Tecnologías utilizadas

- **HTML5**  
- **CSS3 / Tailwind** *(si lo usaste)*  
- **JavaScript**  
- **Git y GitHub** para control de versiones
